# nonstat-amort-latticekrig
This repo contains the code for the article "A Non-stationary, Amortized, Transfer Learning Approach for Modeling Italian Air Quality" by Fusta Moro A., Sirosky A., et al. (2026). We used image-to-image neural architecture to inform and amortize the non-stationary, anisotropic LatticeKrig model, used for modelling Italian air quality.
